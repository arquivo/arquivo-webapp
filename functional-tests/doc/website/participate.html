Participate!

Report Bugs

You can help a lot by using Selenium Grid and reporting problems that you find. You can file any bug you discover using our bug tracking tool.

Join the Community

Hang out in our User Forum. Help new users to get started and veterans to chase a nasty bug. Feel free to also share a new trick or start discussing how we can improve Selenium Grid for the next release...

Spread the Word

Let people know about Selenium Grid. Share your knowledge with your co-workers, your friends or your local Ruby/Java/XP/Pattern group. Why not even blog about it, and tell the whole world!

Express Yourself

If you have a talent for graphics or web design we can use some help. We have plenty of ideas for logos or little cartoons related to Selenium Grid but none of us are up to the task. We also welcome alternative designs for this website (we favor simple and to the point, but we also do not mind esthetics and fun). Please contact us if you feel that you can contribute in that area!

Write Kick-Ass Documentation

Help us improve the existing documentation and create new content. Clear and well written documentation harnessing collective knowledge is a terrific asset for any community. Report English mistakes on the website, create a new page on the wiki, write a blog post or publish an article.

Hack Some Code

We welcome new contributors. The project is still at a very early stage and you can have significant impact. We only ask that you write beautiful code and write tests ;-)
